## 🎲 [Five Roll Defense] / [Yacht Defence]
유니티 타워 디펜스 토이 프로젝트

## 프로젝트 개요
스타크래프트의 유즈맵 'Hell 포커 디펜스'의 디펜스 메커니즘과 '요트 다이스(Yacht Dice)'의 룰을 결합한 타워 디펜스 게임입니다.

## 핵심 시스템
* **족보 기반 타워 생성 :** 다이스 족보 조합에 기반하는 개별 타워 성능
* **시작 미션 / 돌발 미션 및 보상 :** 시작 시 주어지는 미션 / 무작위로 주어지는 미션 성공 여부를 통한 보상 지급
* **찬스 시스템 :** 찬스 시스템을 통한 추가 리롤 기회 제공

## 기술 스택
* **Game Engine:** Unity
* **Language:** C#

### 개발 버전
